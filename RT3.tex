В данном задании было необходимо построить график не линейной функции из вариантов лабораторной работы $№11$, задания $№4$.Следственно в данной работе будет использоваться это уравнение графика $х^3-1,5х^2-2,5х+3=0$. Возьмём интервал $[-2;3]$ с шагом $x=0,1$.


Поскольку проведение касательной не всегда приводит нас к значению корня, так в некоторых случаях касательная наоборот отдаляет нас от его значения, необходимо использовать формулу  $f''*f'=a$. Она показывает в положительной или отрицательной части оси ОХ нам брать точку $x_0$.

Если в уравнении $f''*f'=a$ используется положительный $x_0$:
\begin{enumerate}
  \item $a>0$ следует использовать положительные значения оси ОХ для $x_0$
  \item $a<0$ следует использовать отрицательные значения оси ОХ для $x_0$
\end{enumerate}

После расчёта графика $х^3-1,5х^2-2,5х+3=0$ по формуле $f''*f'=a$ получаем следующее:
\begin{enumerate}
  \item $f'=3x^2 - 3x - 2.5$
  \item $f''=6x-3$
  \item $f''*f'=(6x-3)*(3x^2 - 3x - 2.5)=18x^3 - 27x^2 - 24x +7,5$
\end{enumerate}

Теперь для расчёта того что бы выяснить, из какого множества оси ОХ использовать нам $x_0$ для правильных касательных, нам следует выбрать иксы по всей оси ОХ и подставить выбранные в наше получившиеся уравнение. 
Для построения касательной были выбраны $x_0={-1.3,-1,0.5,1.5}.$
Для проверки необходимого диапазона хватит подставить один $x_0$
Подставим $x_0 = 0,5$ и получим $18*(0,5)^3 - 27(0,5)^2 - 24(0,5) +7,5 = 18*(0,125) - 27(0,25) - 24(0,5) +7,5 = 2,25-4,5-12+7,5=-7,25$, следовательно $a<0$ и с этого следует использовать отрицательные значения оси ОХ.

В данной работе представлены два вида графиков:
\begin{enumerate}
  \item график имеет не правильный подбор  $x_0=0,5$ и $x_0=1,5$.
  \item график имеет правильный подбор  $x_0=-1$ и $x_0=-1.3$.
\end{enumerate}
