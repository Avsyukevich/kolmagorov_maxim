\section{Практическая работа №3 Построение графиков функций}
    В данном задании было необходимо построить график не линейной функции из вариантов лабораторной работы $№11$, задания $№4$.
Следственно в данной работе будет использоваться это уравнение графика $"х^3-1,5х^2-2,5х+3=0"$. Возьмём интервал $[-2;2,8]$ с шагом $x=0,1$.

    Так же по заданию к данному графику следует провести касательную. Возьмём интервал $[-1;2]$.
Уравнение уравнение касательной данного графика в точке $x_0=1$ будет иметь вид $-2.5*x+2.5$.
\begin{tikzpicture}
\begin{scope}[scale=1]

\draw[thin, ->] (-7,0) -- (7,0) node[right] {$X$};
\draw[thin, ->] (0,-7) -- (0,7) node[left] {$Y$};

\foreach \x\xtext in
{-2/-2,2/2,-1/-1,1/1,2.8/2.8} %
\draw (\x,0.1) -- (\x,-0.1) node[below] {$\xtext$};


\draw[domain=-2:2.8, smooth, green] plot ({\x},{((\x)*(\x)*(\x))-(1.5*(\x)*(\x))-(2.5*(\x))+3});
\draw[domain=-1:2, smooth, help lines, violet] plot ({\x},{(-2.5*(\x))+2.5});

\end{scope};
\end{tikzpicture}
