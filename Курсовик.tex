Из задания следует, что функция h(x)=f(x)-g(x), поскольку нам известны функции f(x) и g(x) тогда мы с лёгкостью можем представить функцию h(x) в следующем виде:
\begin{center}

$h = ((sqrt(3))*(sin(x))+(cos(x)))-((cos((2*x) + (\frac{\pi}{3})) - 1)$

\end{center}

Алгоритм исследования функции заключается:
\begin{enumerate}
\item Нахождение области определения функции.
\item Исследование поведения функции на границе области определения, нахождение вертикальных асимптот.
\item Исследование функции на четность или нечетность.
\item Нахождение промежутков возрастания и убывания функции, точек экстремума.
\item Нахождение промежутков выпуклости и вогнутости функции и точек перегиба.
\item Нахождение горизонтальных и наклонных асимптот.
\item Вычисляем значения функции в промежуточных точках.
\item Построение графика.
\end{enumerate}

Поскольку математический пакет "Scilab" работает с присвоением числа, а не разом с рядом чисел, рассчитать область определения функции не является возможным. Данный расчёт придётся провести в ручную.Исходя из того, что область определения функции определяется из условий:

\begin{itemize}
\item Деление на выражение с переменной, или присутствие переменной в знаменателе дроби ($y=x+\frac{2x}{(x^4)-1}$).Это связано с тем, что на нуль делить нельзя.
\item Переменная под знаком корня четной степени или в показателе корня, к примеру,$y=\sqrt(x+1)$, $y=\sqrt[x]{2^{3x-3}}$. В основе лежит тот факт, что корень четной степени имеет смысл лишь тогда, когда число под ним неотрицательно. А что касается показателя корня, то им может быть лишь какое-нибудь натуральное число, отличное от единицы.
\item Переменная в основании степени с отрицательным или не целым показателем. Вот примеры таких функций$y=5(x+1)^{-3}, y=-1+x^{1\frac{1}{3}}, y=((x^3)-x+1)^{\sqrt(2)}$
\item Наличие переменной под знаком логарифма или в основании логарифма, например, $y=\ln\frac{x^2+x}{4}, y=1+\log_{x-1}(x+1)$. Здесь учитываем, что в основании логарифма должно быть положительное и отличное от нуля число, а под знаком логарифма положительное число.
\item Наличие переменной под знаком тангенса или котангенса, $y=x^3+tg(2x+5), y=ctg(3x^3-1)$. Это и понятно, ведь тангенс и котангенс имеют смысл не для любого действительного числа.
\item Наличие переменной под знаком арксинуса или арккосинуса, $y=arcsin(x+2)+2x^2, y=arccos(|x-1|+x).$ Тут учитываем, что арксинус и арккосинус мы определяли для чисел из отрезка от ?1 до 1, а не для любого числа
\end{itemize}

Поскольку формула, которой  функции h(x), не содержит ни одного признака из представленного списка, то  область определения функции есть множество вещественных чисел (R). Следственно в данном случае $x \epsilon R$

Для нахождение промежутков возрастания и убывания функции, точек экстремума. Необходимо провести решение неравенств h'(x)>0 и h'(x)<0. Точки, в которых производная обращается в ноль, называют стационарными. Критическими точками функции называют внутренние точки области определения, в которых производная функции равна нулю или не существует.

Таким образом, чтобы определить промежутки возрастания и убывания функции:
\begin{itemize}
\item Необходимо найти производную функции h'(x)
\item Высчитываем критические точки. Точки при которых выражение не может существовать.
\item Расставляем область определения критическими точками на интервалы по числовой прямой.
\item Проводим определение знака производной на каждом из промежутков.
\end{itemize}

Для данных расчётов в математическом пакете "Scilab" нам понадобиться использовать создание собственной функции используя команды function и endfunction. Для этого мы создаём зависимость y от x командой function у = myfunction (x), затем создаём нагрузку на переменную x обычной командой присвоения командой y=x*n, после чего закрываем раздел функции для её создания с помощью команды endfunction.

Листинг данной операции:

$-->$endfunction

\item атем проводят уточнение отделенных корней, или, иначе говоря, находят корни с заданной точностью.

\item Разбиваем область определения полученными точками на интервалы;
\item Определяем знак второй производной на каждом из промежутков. Знак «плюс» будет соответствовать промежутку вогнутости, знак «минус» - промежутку выпуклости.
